# To Create a New Page

1. Copy the [`CopyMe.html`](CopyMe.html) template to the 
   [`Pages`](../Pages) folder.
2. Edit the copied template to customize it.
   * Filename
   * HTML `lang`
   * HTML `title`
   * HTML `textarea`
